# Global Store Return Prediction and Business Data Analysis

## Information

Used Global Store retail order data to turn product return status into a binary prediction task and analyse how sales, profit, discount, shipping cost, delivery mode, customer features, and product categories relate to return behaviour.

Cleaned and prepared data in Python, built return-rate features by delivery mode, customer, customer segment, city, region, category, and sub-category, and trained Logistic Regression and MLPClassifier models.

Evaluated model performance using Accuracy, Precision, Recall, F1 Score, and ROC-AUC. The project helps retailers identify high-return orders and improve discount, logistics, and customer management decisions.

## Tools

Python, Pandas, Scikit-learn, Logistic Regression, MLPClassifier, Matplotlib
