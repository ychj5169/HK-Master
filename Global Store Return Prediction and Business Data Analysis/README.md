# AI-Based Product Return Prediction and Image Clustering

---

## Course Information

**Course:** INFS3822 – Artificial Intelligence for Business Analytics  
**Institution:** UNSW Business School

---

## Project Overview

This project focuses on applying machine learning techniques to analyse product return behaviour using structured transaction data and unstructured product image data. The project includes data preprocessing, feature engineering, supervised learning model development, image clustering, model evaluation, and business insight generation to support operational decision-making.

---

## Programming Language

- Python

---

## Dataset

- Structured transaction dataset (`A1 - global_store.csv`)
- Product image dataset (`A1 - product_images.zip`)

---

## Software

- Google Colab

---

## Machine Learning Models

- Logistic Regression
- Deep Neural Network (MLPClassifier)
- K-Means Clustering

---

## Project Workflow

- Data preprocessing and feature engineering
- Structured data modelling using supervised learning
- Model evaluation using confusion matrix, ROC curve, accuracy, precision, recall, F1-score and ROC-AUC
- Product image feature extraction
- Image clustering using K-Means
- Business insight interpretation
